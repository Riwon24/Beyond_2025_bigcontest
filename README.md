# 🤖 AI 마케팅 전략 코치 (2025 빅콘테스트 AI데이터 활용분야)

AI 기반으로 우리 매장을 분석하고, **경쟁 매장과의 비교**를 통해 맞춤형 마케팅 전략을 자동으로 추천해주는 **전략 코치 챗봇 시스템**

---

## 🧠 핵심 기능

- 📌 **매장명 입력** 시 자동 분석
- 🔍 **경쟁 매장 평균 대비** 배달률/재방문율/신규고객률 분석
- 🧾 **주 고객층 / 유입 필요 타겟 / 상권 유형** 자동 판별
- 💡 **전략 카드 + 마케팅 문구** 자동 생성 (룰 기반 + Gemini LLM)
- 📊 **퍼센타일 그래프**로 시각화 제공 (Plotly)
